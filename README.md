## Before you submit — 2 things to personalize
1. **GitHub username:** In `index.html`, `about.html`, and `resume.html`, find
   `https://github.com/jacem1303` (in the footer) and replace `jacem1303`
   with your real GitHub username.
2. **YouTube video (optional):** In `scratch.html`, the embed uses a placeholder video.
   To use your favorite golf video: open it on YouTube → Share → Embed → copy the ID
   that comes after `/embed/`, and paste it in place of the current ID in the iframe `src`.

## Host it on GitHub Pages
1. Create a new **public** repo named exactly `jacem1303.github.io`
2. Upload all of these files (keep the `css/` and `img/` folders intact)
3. Repo → **Settings → Pages** → Source = **main** branch, root → Save
4. Your site goes live at `https://jacem1303.github.io/` in about a minute
