and a single-page wealth-growth app.

## Files (all flat — no folders, no image files)
- index.html – home page (your photo is embedded right in the file)
- about.html – about page
- resume.html – HTML resume
- scratch.html – from-scratch golf page (golf graphic is drawn inline)
- app.html – wealth-growth simulator (single self-contained page)
- style.css – styles for the professional pages
- scratch.css – hand-written styles for the golf page

## Re-uploading to GitHub (the important part)
Your repo already exists at jacem1303.github.io. To replace the old files:
1. Open your repo -> Add file -> Upload files
2. Drag in ALL of these at once:
   index.html, about.html, resume.html, scratch.html, app.html, style.css, scratch.css
   (No folders and no images to worry about this time.)
3. Click Commit changes. GitHub overwrites the old files.
4. If the old repo still has an img or css folder from last time, you can delete them.

Your site updates at https://jacem1303.github.io/ within a minute.
